===============================
 LOCAL RUN STEPS  (git)
===============================

1) Go to project folder
--------------------------------
cd PassSecure


2) Create virtual environment
--------------------------------
python -m venv venv


3) Activate environment (Git Bash)
--------------------------------
source venv/Scripts/activate


4) Install dependencies
--------------------------------
pip install flask zxcvbn


5) Run the app
--------------------------------
python app.py


6) Open in browser
--------------------------------
http://127.0.0.1:5000
